# Android 视频播放器项目 (VMPlayer)

## 项目简介

这是一个基于 Android 平台开发的**实验性**
视频播放器项目，是自己对视频播放器渲染技术深入探索的成果。项目支持多视频片段播放、实时预览、视频导出等功能，采用模块化架构设计，包含自定义的媒体处理框架和播放器实现。

> **注意**: 这是一个个人学习和技术探索项目，主要用于研究视频渲染、OpenGL ES、音视频同步等技术。代码仅供学习参考，不建议直接用于生产环境。

## 技术探索重点

本项目重点探索和实践了以下技术领域：

- 🎬 **自定义视频渲染管线**: 基于OpenGL ES构建完整渲染流程
- 🎵 **音视频同步机制**: 精确的时间轴管理和同步算法
- ⚡ **多线程架构**: 解码、渲染、音频播放的线程协调
- 🔄 **内存管理优化**: 纹理缓存、解码器资源管理
- 📤 **视频导出技术**: MediaCodec编码和文件输出

## 功能特性

### 核心功能

- ✅ **多视频片段播放**: 支持播放多个视频片段组成的播放列表
- ✅ **实时视频预览**: 基于 OpenGL ES 的高性能视频渲染
- ✅ **视频导出**: 支持将播放列表导出为单个视频文件
- ✅ **播放控制**: 播放、暂停、停止、进度控制、循环播放
- ✅ **时间轴管理**: 精确的时间轴管理和片段切换
- ✅ **音视频同步**: 音频和视频的精确同步播放
